# Summer School 2026 — Lessons

Seven lessons, ~30 min each (one may run to 45 if the discussion is good — that's a win, not a problem). Each one has a **solo** part and a **with-Oma/Opa** part. Some days are heavier, some are lighter — alternate on purpose.

## Suggested weekly rhythm

| Day | Lesson | Type |
|---|---|---|
| Mon | [`symbols/`](symbols/lesson.md) — *The # is actually two letters* | Light, curious — easy on-ramp |
| Tue | [`german-aura/`](german-aura/lesson.md) — *Modern German slang* | Fun, song-based |
| Wed | [`sigmoids/`](sigmoids/lesson.md) — *Why nothing grows forever* | Math muscle |
| Thu | [`ai-society/`](ai-society/lesson.md) — *Is AI exposing an old problem?* | Discussion-heavy |
| Fri | [`art-imagine/`](art-imagine/lesson.md) — *Make something* | Creative reset |
| Mon | [`german-schrei/`](german-schrei/lesson.md) — *A love song to a bully* | Deeper German, big topics |
| Tue | [`ai-truth/`](ai-truth/lesson.md) — *The sparkle on every button* | Capstone discussion — pairs with Thursday's AI lesson |

## Lesson shape (consistent across all 7)

- **Hook** — 1-line "why care"
- **Warmup** — solo, 2–5 min
- **Watch / Read / Listen** — exact timestamps for video segments
- **Vocabulary** — words worth learning
- **Solo task** — write/draw/sketch (~10 min)
- **Discussion with Oma/Opa** — open-ended questions, no right answers
- **Stretch** — bonus if she's hooked

## What you'll need on hand

- Paper, pen/pencil, two highlighters
- A device for the videos (`videos/` folder) and the two web links (sigmoids article, imaginemore.art)
- Graph paper for the sigmoids lesson

## Notes

- All video files are in `../videos/`, lyrics in `../lyrics/`.
- Lessons 3 (`ai-truth`) and 4 (`ai-society`) are designed to pair — doing them on the same week lets the second one build on the first.
- The two German lessons are not back-to-back: *Aura* (modern, slangy) warms the ear before *Schrei nach Liebe* (richer vocabulary, heavier topic).
